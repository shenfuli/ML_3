## 数据格式
https://www.datafountain.cn/competitions/277/details/data-evaluation <br>
比赛中提供的训练数据为多行文本，每一行分为四列，使用\t分割，第一列为文档ID, 第二列为案件事实描述，第三列为罚金额度类别，第四列为对应的法律条文编号序列，其中法律条文编号序列是有”,”号分割。
